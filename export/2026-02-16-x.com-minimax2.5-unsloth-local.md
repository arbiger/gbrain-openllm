## How to Run Locally (Mac mini)
1. **Install `llama.cpp`**:
   ```bash
   git clone https://github.com/ggml-org/llama.cpp
   cmake llama.cpp -B llama.cpp/build -DGGML_CUDA=OFF
   cmake --build llama.cpp/build --config Release --target llama-cli
   ```

2. **Download the quantized GGUF**:
   ```bash
   pip install huggingface_hub hf_transfer
   hf download unsloth/MiniMax-2.5-GGUF --local-dir ./models --include "*UD-Q3_K_XL*"
   ```

3. **Run inference**:
   ```bash
   ./llama.cpp/llama-cli \
       --model ./models/UD-Q3_K_XL/MiniMax-2.5-UD-Q3_K_XL-00001-of-00003.gguf \
       --jinja \
       --temp 1.0 \
       --top-p 0.95 \
       --top-k 40 \
       --ctx-size 16384 \
       --fit on
   ```
