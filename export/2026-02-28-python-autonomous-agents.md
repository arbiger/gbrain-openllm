# Learn Python and Build Autonomous Agents

## 摘要

freeCodeCamp.org 發布了一個新課程，从 Python 基礎到 AI Agent 開發。

## 課程大綱

### Module 1: Python 基礎
- 變數、循環、函數、模組

### Module 2: 數據科學基礎
- NumPy、Matplotlib、Pandas、SQLite

### Module 3: APIs & 部署
- REST APIs 工作
- 使用 FastAPI 構建

### Module 4: AI 前沿
- LLM 整合
- Open Source models (HuggingFace)
- Agent Tools

## 重點
- 使用 Open Source 模型（HuggingFace）
- 建構 AI 與世界互動的橋樑
- 從 Python 到自主 AI Agent 的完整技術棧
