# 2026年3月美股市場分析與基本面篩選方法

## 影片重點

### 市場概況
- 2026年3月初Nasdaq 100期貨上演大反彈，收復前日全部跌幅
- 伊朗戰爭對股市影響有限，呈現「跌了個寂寞」走勢
- 軟體板塊整體走強，IGV 連續兩周收紅

### 影片核心：基本面篩選被低估股票的方法

#### 篩選條件
- 市值：Mega Cap (2000億+)
- 交易所：NASDAQ
- 產業：半導體 (Semiconductor)
- 國家：美國

#### 評估三個指標
1. **Forward P/E (前瞻本益比)** - 與同業平均比較
2. **Trailing P/E vs Forward P/E 差異 (Divergence)** - 隱含成長率
3. **PEG Ratio** - 本益比相對於成長率

#### 實際案例：半導體五大天王
| 股票 | Forward P/E | Trailing P/E | PEG Ratio |
|------|-------------|--------------|-----------|
| MU (美光) | 8.8 | - | - |
| NVDA (輝達) | 16.76 | 37.23 | - |
| AMD | 18.23 | - | - |
| AVGO (博通) | 22.1 | - | - |
| INTC (英特爾) | 47.55 | - | 1.03 |

#### 計算隱含成長率
公式：(Trailing P/E ÷ Forward P/E - 1) × 100%

以 NVDA 為例：
- 使用 Yahoo Finance 數據：37.23 ÷ 22.17 - 1 = 68% EPS成長預期
- 分析師預期未來12個月 EPS 成長約68%

#### 篩選邏輯
1. 計算同業平均 Forward P/E ≈ 22.69
2. 篩掉高於平均的股票 (INTC 47.55 → 剔除)
3. 找出最大 Divergence → 隱含成長率最高者
4. PEG < 1 表示市場對未來成長預期強

### 重點結論
- **NVDA**: Forward P/E 16.76 (低於平均22.69)，隱含68% EPS成長 → 相對低估
- **MU**: Forward P/E 8.8，最低估
- **INTC**: Forward P/E 47.55，PEG 1.03 → 篩掉
- 整體半導體板塊從未來獲利角度看，仍屬相對低估

### 投資思路
- 不要只看股價創新高，要看**未來獲利能力**對應的估值
- 結合基本面 + 技術面 + 風險管理
- 利用短期波動找到更好的進場時機
