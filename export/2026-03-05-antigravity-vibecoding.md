# Antigravity 教學 - Vibe Coding 開發工具

> **TLDR:** 介紹 Google AI Studio 的 Antigravity 工具，如何進行 Vibe Coding（用自然語言開發程式），包含規則設定、自動化流程、MCP、Skills 等功能。

---

## 重點整理

### 1. 為什麼用 Antigravity？
- AI 驅動的網頁開發工具
- 可免費使用（額度較少）
- Pro 會員 5 小時更新一次，免費版一週更新一次

### 2. 必裝工具
- **Antigravity** - 主程式
- **Git + GitHub CLI** - 版本控制
- **Python + Node.js** - 執行腳本
- **PanDoc** - 格式轉換
- **FFmpeg** - 影音處理
- **YT-DLP** - 影片下載

### 3. 核心功能

#### 規則 (Rules)
- **Global 規則**：全域設定（繁體中文輸出、環境如 Windows、版本控制）
- **Workspace 規則**：專案特定規則

#### 自動化流程 (Workflow)
- 可順序執行的提示詞
- 範例：部署網站到 GitHub Pages

#### MCP (Model Context Protocol)
- 讓 AI 連結外部工具的標準協議
- 範例：Context7（文件查詢）、Notion、資料庫

#### Skills
- Anthropic 推出的新功能
- 有點像 MCP + Workflow 的結合
- 可安裝額外技能（已有 8 万多個 Skills）
- 網站：skillsmp.com

### 4. Agent Manager
- 可讓 Antigravity 多工執行
- 任務在背景執行，不阻塞主線程
- 有 Playground 可執行臨時任務

### 5. Artifacts
- 追蹤任務進度
- 產出的檔案、圖片、結案報告
