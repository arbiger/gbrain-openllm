## 夜間記憶 (22:00-23:25)

### Anthropic 完整交接系統
- 來源：Obsidian 筆記《Long-running Agent 的真正意思：交接零成本》
- 核心概念：Long-running Agent 不是一個 session 跑很久，是交接零成本
- 關鍵原則：每個 session 結束時代碼必須是「可以 merge 到 main」的狀態

#### 七步迴圈（每個 session）
1. 讀交班日誌 + git log
2. 讀 feature list，揀最高優先
3. 跑 init.sh 起動 dev server
4. Smoke test 確認 app 活著
5. 實作一個 feature
6. E2E 驗證這個 feature 真的 work
7. git commit + 更新交班日誌

#### 多層 Review Stack
- /simplify — 三個平行 agent 看 code reuse / quality / efficiency
- /review — PR-level review，找 bugs、logic errors
- /qa — 用瀏覽器做 E2E 測試

#### Squad 整合方向
- 每個 phase 寫 findings.md、progress.md、handover.md
- 交接時確保 code 可 merge
- squad-manager 統一整理 Wrap-up
