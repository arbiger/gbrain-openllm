# Note-195056

慈濟（ Tzu Chi）組織與財務結構研究

## 會員月費
- 一般會員：50-100元/月（各人隨份隨力）
- 年費：約600-1,200元

## 委員晉升之路（以拉下線為主）
| 階級 | 必要條件 |
| 見習委員 | 15戶下線 |
| 培訓委員 | 25戶下線 + 一年培訓 |
| 授證委員 | 40戶下線 + 授證 |

## 制服費用（隱藏成本）
- 基礎制服全套：數萬元
- 法船別針：8,000元
- 升級累計：數十萬

## 榮譽董事（榮董）— 100萬門檻
- 直接榮董：一年內捐滿100萬
- 委員榮董：先當委員再捐100萬（俗稱三朵花）
- 福利：金證、金徽章、表揚大會、讀書會、免費健康檢查（市價3,500-16,000元）
- 免費健康檢查內容（對標公務人員方案）：
  - 基本：身高體重血壓、尿/糞便、X光、心電圖、腹部超音、血液檢查
  - 升級：無痛胃鏡、大腸鏡、低劑量肺部CT、各科會診

## 慈濟醫院就醫費用
- 掛號費：門診150元、急診220元
- 西醫門診部分負擔：420元（醫學中心等級）
- 與一般醫學中心費用相同，無特殊優惠

## 爭議點
1. 用下線戶數決定階級
2. 制服變相收費
3. 功德款制度
4. 國外賑災被質疑為國際知名度
